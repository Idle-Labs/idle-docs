# AATranche

This method returns the address of the AA (Senior) tranche token contract (ERC20).

### Function

```solidity
function AATranche(
) external view returns(address);
```

### Return values

| Value          | Description                                       |
| -------------- | ------------------------------------------------- |
| `address _val` | Address of the AA Tranche token contract (ERC-20) |
